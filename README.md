# Nasdaq ETP Data Quality & AUM Monitoring Dashboard

This project simulates a real-world data quality audit for Exchange Traded Products (ETPs), focusing on **AUM, NAV, and regional completeness**.

The goal is to identify operational data gaps and provide an interactive Excel dashboard for portfolio monitoring.
